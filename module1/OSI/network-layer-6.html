Network Layers of OSI Model

Application Layer (layer 7)

The range of this stuff include resource sharing, remote file access, remote printer access, network management, and electronic communication (email).

There is an outsized range of application layer protocols that are familiar to the common internet user, as well as the File Transfer Protocol (FTP), Simple Mail Transfer Protocol (SMTP), Domain Name Service (DNS), and Hypertext Transfer Protocol (HTTP).
