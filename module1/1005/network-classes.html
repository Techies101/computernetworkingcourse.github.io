Network Classes

The InterNIC, the organization that manages the Internet, assigns Internet addresses. These IP addresses are classified into groups. A, B, and C are the most ordinary classes. Although classes D and E exist, they are rarely used by end users. Each address class has a unique default subnet mask. The first octet of an IP address can be used to determine its class. The following are the Class A, B, and C Internet address ranges, along with an example address:

Class A networks have a default subnet mask of 255.0.0.0 and a first octet of 0-127. The IP address 10.10.10.1 belongs to the class A network. Its first octet is 10, which ranges from 1 to 126.

Class B networks have a default subnet mask of 255.255.0.0 and a first octet of 128-191. The address 172.50.20.0 is classified as a class B address.It has a first octet of 172, which is between 128 and 191.
