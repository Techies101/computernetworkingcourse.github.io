Subnetting

The primary IP address is 192.168.1.0/24, which refers to the 24 displayed subnet masks. As a result, the first subnet mask is made up of three octets of 1's and one octet of 0's. If you want to divide this network into four subnets, for example, you must calculate the bits that will be borrowed in the subnet mask. There's a formula for this: 2^m=subnet.

Two bits are required to replace it. We can see that the last octet's two bits are one if we take these two bits from the subnet mask. This is my network's new subnet mask. We get 27 + 26 = 192 when we convert the subnet mask from binary to base decimal using one-bit places. As a result, the new subnet mask for my network is 255.255.255.192.
