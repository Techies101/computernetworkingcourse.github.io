IPv4

Every host on a single network has the same network address. Each host also has an address that identifies it uniquely. The address is either globally or locally unique, depending on the scope of the network and the type of device. Devices visible to users outside, for example, web servers must have a globally unique IP address to be visible to users outside the network.

The Internet Assigned Numbers Authority (IANA) is a centralized numbering authority that assigns IP addresses (IANA). IANA ensures that addresses are globally unique where possible and maintains a large address space for use by devices that are not visible outside of their own networks.
