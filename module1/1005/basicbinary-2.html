Basic binary

It's the same as counting in decimal, except we get to 10 much faster.

How do we count in Decimal? Begin at 0 and count 1,2,3,4,5,6,7,8 & 9. This is the decimal system's final digit. So we start from 0 again, but this time add 1 to the left.

In binary, the same thing occurs.
