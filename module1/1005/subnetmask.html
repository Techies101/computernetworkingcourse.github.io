Subnet mask

This number has no obvious meaning unless you know that 255 in binary notation equals 11111111. The subnet mask is thus 11111111.11111111.11111111.0000000. The network and host portions of the address can be separated by lining up the IP address and the subnet mask:

11000000.10101000.01111011.10000100-IP address (192.168.123.132) 11111111.11111111.11111111.00000000-Subnet mask (255.255.255.0)

The network address is identified as the first 24 bits (the number of ones in the subnet mask). The host address is identified by the last 8 bits (the number of remaining zeros in the subnet mask). It gives you the following addresses:

11000000.10101000.01111011.00000000-Network address (192.168.123.0) 00000000.00000000.00000000.10000100-Host address (000.000.000.132)
